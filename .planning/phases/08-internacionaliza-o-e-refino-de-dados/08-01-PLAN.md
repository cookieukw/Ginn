# Plan 08-01: Internacionalização e Refino de Dados

**Objective**: Refatorar o código e a estrutura de dados para utilizar chaves em Inglês (boas práticas), auditar a precisão biológica dos animais atuais e expandir a base de dados com novas espécies.

<must_haves>
- [ ] Refatoração das chaves do objeto `weights` e do JSON para Inglês (ex: `features` em vez de `Características`).
- [ ] Correção de erros gramaticais e conceituais nos dados (ex: `Oviparidade` -> `Ovíparo`).
- [ ] Adição de pelo menos 10 novos animais à base de dados.
- [ ] Atualização do `DataManager` para ler o novo formato do JSON.
- [ ] Atualização do `DialogueManager` para mapear os novos nomes das propriedades em Inglês de volta para as perguntas em Português.
</must_haves>
